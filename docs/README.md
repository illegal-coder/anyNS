# anyNS 文档目录

本目录是 anyNS 的需求与交付文档集。anyNS 定位为基于 PowerDNS 的去中心化 NS 解析与 DNS 安全平台，同时兼容传统 ICANN 公网 DNS、AdGuard/审计 DNS、DNSLog 蜜罐联动和一站式部署。

## 核心决策

- PowerDNS 是主解析链路和去中心化插件承载层。
- CoreDNS 仅作为低优先级可选安全增强组件，不承载去中心化解析主逻辑。
- 系统必须正常对接 ICANN 传统公网域名解析，不能作为孤立替代根服务独立运行。
- `WALLET` 是 DNS RR 类型，按 IANA type 262 处理，不作为 `.wallet` 后缀处理。
- 部署交付必须同时覆盖本机一键脚本和 Docker Compose。
